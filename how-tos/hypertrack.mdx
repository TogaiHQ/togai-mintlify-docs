[Hypertrack](https://hypertrack.com/) is a live location platform in the cloud offering developer tools for developers. Their [pricing model](https://hypertrack.com/pricing) is usage based and directly dependent on the type of order tracking chosen per order.

You can view how this is easily configurable in Togai below. The configuration is done by Sajed and Adhitya, before the song [Around the world](https://www.youtube.com/watch?v=K0HSD_i2DvA) finishes.
